# Rock-Paper-Scissors-Game

An Interesting Game of Rock, Paper and Scissors with HTML, CSS and JavaScript which Identifies Computer Moves and Player Moves and Declares a Winner based on Moves.

## ✨ Features

- **Interactive Gameplay**: Play against a computer opponent with random move generation
- **Score Tracking**: Keep track of wins, losses, and ties
- **User-Friendly Interface**: Simple and intuitive design built with HTML and CSS
- **Real-time Feedback**: Instant game results with visual feedback
- **Responsive Design**: Works seamlessly across different devices

## 🚀 Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- Git installed on your machine

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Srinathi762/Rock-Paper-Scissors-Game.git
   cd Rock-Paper-Scissors-Game
   ```

2. Open the game in your browser:
   - Simply double-click the `index.html` file, or
   - Right-click and select "Open with" your preferred browser

## 🎮 Usage

1. Open `index.html` in your web browser
2. Click on your choice: **Rock**, **Paper**, or **Scissors**
3. The computer will automatically make its choice
4. View the result of the round
5. Your score and the computer's score will be updated
6. Continue playing and try to beat the computer!

## 📖 How to Play

The game follows classic Rock-Paper-Scissors rules:

- **Rock** ✊: Crushes Scissors
- **Paper** ✋: Covers Rock
- **Scissors** ✌️: Cuts Paper

## 🎯 Game Rules

- Each round, both the player and computer make a move simultaneously
- The winning move beats the opponent's move according to the rules above
- If both players choose the same move, it's a tie
- Ties do not award points to either player
- The game continues indefinitely until you decide to stop or refresh the page
